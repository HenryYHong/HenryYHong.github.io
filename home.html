ASL Language Classifier

A machine learning model that classifies American Sign Language gestures.

This project focused on creating a robust machine learning model to classify American Sign Language (ASL) gestures, leveraging both convolutional neural networks (CNNs) and random forests to achieve accuracies of 80.3% and 97.2%, respectively. To overcome the challenges of limited pre-existing datasets, I generated my own training data by capturing real-life images of ASL gestures, ensuring model generalizability. The project utilized MediaPipe for hand landmark detection, followed by advanced feature extraction for training the models. The classifier demonstrated real-time gesture recognition capability, making it highly applicable for live ASL translation systems.

  • Tech: Python • TensorFlow • MediaPipe
  • Link: GitHub Repository
  • Presentation: Slides
  • Date: Fall 2024

Music Genre Classification

Exploring machine learning for audio classification.

Principal Component Analysis performed on the 10 genres.

Our team analyzed a dataset of 1,000 audio files across 10 genres to classify music genres using various machine learning models. By leveraging Random Forest, K-Nearest Neighbors, and Support Vector Machines, we refined our approach to achieve a highest accuracy of 80.6%. Additionally, neural networks were employed for pattern recognition, enhancing classification precision.

First tree of our trained Random Forest model.

While addressing challenges such as outlier detection and confusing column names, we improved our understanding of machine learning and audio signal processing. The project showcases the power of statistical modeling and neural networks in complex data classification tasks.

  • Tech: Python • scikit-learn • librosa • TensorFlow
  • Links: Deepnote, Slides
  • Date: Fall 2023

    • Deep Oculos

    Second-Place Winner, Berkeley's 2023 Annual Datathon for Social Good

    Deep Oculos leverages Generative Adversarial Networks (GANs) to create synthetic retinal eye motion videos and Long Short-Term Memory (LSTM) networks for disease-related eye motion trace prediction. This project was developed to aid C. Light Technologies in disease modeling and diagnostics by simulating patient-specific retinal movements.

    Using a four-layer LSTM with dropout regularization, we achieved a 93.4% accuracy in predicting disease-related pupil movements. Our GAN generated synthetic videos, starting at 64x64 resolution and upscaled to 512x512, embedding pupil trace statistics directly into the frames. This innovative approach earned our team the 2nd Place award at UC Berkeley’s 5th Annual Datathon for Social Good.

    • Tech: TensorFlow • Keras • Scikit-Learn • Seaborn • Pandas
    • Links: GitHub, Slides
    • Date: Winter 2023

    Fullstack DSS Project

    An AI-powered member directory with personalized recommendations for the Data Science Society.

    General member directory showcasing all members of the Data Science Society.

    Interactive member profile with GPT-powered learning and friend recommendations via a Graph Neural Network.

    This project introduced an AI-assisted member directory for over 100 students and alumni, developed using React and Flask and deployed on Heroku. It featured a Graph Convolutional Network (GCN) implemented with PyTorch, achieving an ROC AUC score of 0.72 in link prediction accuracy between members. Secure login was integrated via Google OAuth 2.0, and data was retrieved dynamically through Airtable API. Additionally, responses were generated using OpenAI API, enhancing engagement and interaction within the directory.

    • Tech: React • Flask • PyTorch • Google OAuth 2.0 • OpenAI API
    • Links: GitHub, Midterm Slides, Final Slides
    • Date: Spring 2024

    WordNet Integration

    Developing an enhanced browser-based tool to explore word relationships using advanced data structures.

    This project involved creating a dynamic interface to explore word relationships by integrating the WordNet semantic lexicon into a web-based application. I implemented core features such as hyponym traversal and graph-based word relationships using advanced Java data structures, including custom-designed directed graphs and time-series data mapping. The tool provided a visual representation of word hierarchies and historical usage trends through an interactive front end.

    The backend, built from scratch, supported efficient query handling for datasets with millions of entries, showcasing my expertise in algorithmic design and optimization. Key challenges addressed included parsing messy datasets and ensuring scalable graph traversal, solidifying my understanding of real-world data processing.

  • Tech: Java • Directed Graphs • Breadth-First Search • WordNet API • JUnit
  • Link: Project Details
  • Date: Fall 2023

    • Flappy Bird Extra Hard

    A challenging and modern twist on the classic Flappy Bird game.

    In this enhanced version of the popular game, I introduced new mechanics to elevate the challenge and excitement. Players navigate a bird through procedurally generated pipes with randomized gap sizes, while avoiding rocket-like enemies that require quick reflexes and precise timing. With textured ground, animated clouds, and dynamic gameplay elements, this project pushes the limits of classic game design.

    Key features include:

    • Dynamic Pipe Gaps: Randomized gap sizes add unpredictability to each level.
    • Enemy Rockets: Fast-moving obstacles that test players’ reflexes and strategic thinking.
    • Polished Visuals: Textured ground, animated clouds, and an upscaled bird sprite for a modern look.
    • Screenshot Capture: Save gameplay screenshots by pressing 'P' for easy sharing.

    This project showcases my ability to integrate physics-based gameplay, design aesthetic enhancements, and implement new gameplay elements in Python. Check out the full code on GitHub for more details.

  • Tech: Python • Pygame • Custom Physics Engine
